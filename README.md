 Sistema CPaaS - Telecall (Front-End)

📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de um sistema web front-end inspirado na plataforma **CPaaS (Communication Platform as a Service)** da Telecall.

O sistema permite:

* Cadastro de usuários com validação completa
* Autenticação (login)
* Navegação entre serviços
* Simulação de funcionalidades CPaaS
* Persistência de dados utilizando LocalStorage

---
 🎯 Objetivo

Atender aos requisitos propostos na disciplina de Front-End, aplicando conceitos de:

* HTML5
* CSS3
* JavaScript
* Responsividade com Bootstrap
* Experiência do usuário (UX/UI)

---

 🧩 Funcionalidades

 🔐 Cadastro de Usuário

* Validação de todos os campos obrigatórios
* Nome com 15 a 60 caracteres
* Login com 6 letras
* Senha com 8 letras
* Validação de telefone e CPF
* Confirmação de senha
* Feedback visual ao usuário (sem uso de alert)

---

 🔑 Login

* Autenticação baseada em dados do LocalStorage
* Redirecionamento para área logada
* Exibição do usuário no topo da tela

---

 🏠 Tela Principal

* Menu com serviços CPaaS:

  * 2FA
  * Número Máscara
  * Google Verify Calls
  * SMS Programável
* Interface moderna e responsiva

---

 📄 Telas Internas

* Exibição dinâmica de conteúdo por serviço
* Submenu funcional
* Layout padronizado em todas as páginas

---

 ♿ Acessibilidade

* Modo escuro (dark mode)
* Aumento e diminuição de fonte

---

 🎨 Design

* Identidade visual inspirada na Telecall
* Interface moderna estilo SaaS
* Componentes com animações e feedback visual

---

 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Bootstrap 5

---

 💾 Armazenamento

* Utilização do **LocalStorage** para:

  * Dados de usuário
  * Sessão de login

---

 📱 Responsividade

O sistema foi desenvolvido com foco em:
